## Infinite Tic-Tac-Toe

A dynamic Tic-Tac-Toe game with an infinite board concept. Players can make moves as usual, but as the board fills up, older moves are automatically removed to maintain fluid gameplay. Built with HTML, CSS, and JavaScript, the game provides an engaging twist on the classic game.

### Features
- Infinite board with smooth gameplay.
- Automatic removal of older moves as the board fills up.
- Responsive design for various devices.
- Highlights winning combinations.

### How to Play
1. Take turns placing your mark (X or O).
2. Stay strategic, as older moves will be removed when the board fills up.

### Technologies Used
HTML, CSS, JavaScript
